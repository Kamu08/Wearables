## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/wearables.git
    ```

2. **Navigate to the project directory:**
    ```bash
    cd wearables
    ```

3. **Install dependencies:**
    ```bash
    npm install
    ```

4. **Start the development server:**
    ```bash
    npm start
    ```

## Setting up Stripe Integration
